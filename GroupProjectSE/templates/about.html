About This Project

This prototype web application was created as part of the Introduction to Software Engineering course.

It demonstrates a collaborative effort in developing a simple, data-driven Flask web app with a clean interface and maintainable structure.

Project Overview

Our application explores the integration of web technologies, Python, and dataset interaction within Flask. The project serves as a foundation for future functionality such as data uploads and machine learning model integration.
